# Midterm-Project_Pristine
This is a repo I made for a game I'm working on as a midterm project in my Game Programming class, I've called it "Pristine", legally distinct from the Stephen King story "Christine". You play as the titular haunted car, chasing down a cretin that has insulted you as he tries to escape the city.
